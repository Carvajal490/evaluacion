# **Guía técnica de navegacion creacion en git bash**   
En este proceso se puede ver como navegar crear carpetas y archivos y volver tu carpeta un repositorio

## **Comandos**  
- cd "archivo o carpeta"  navegar en el pc  
- mkdir "Nombre carpeta"  crear carpeta  
- ls   listar archivos carpetas disponibles   
- Touch "nombre" crear archivo  

### Crear repositorio*  
comando:   
- git init  
depues habilitar la opcion del escritorio de ver documentos oculto si aparece esta bien
